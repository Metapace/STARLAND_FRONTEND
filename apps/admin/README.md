# React-arco-admin

node 版本需要在 v14 之上

```bash
pnpm i

pnpm sta

```

## 该系统技术栈特点

- 基于 webpack5.x 配置
- eslint 严格代码规范
- prettier 统一代码风格
- husky 代码提交检测
- 环境变量配置规范化

## 技术框架

- [x] react18
- [x] react-dom
- [x] react-router-dom 6.x
- [x] react-hook-form
- [x] hooks
- [x] arco-design

## 代码规范

- [x] eslint
- [x] prettier
- [x] husky

代码 x 层面需要对自己严格要去的规范
